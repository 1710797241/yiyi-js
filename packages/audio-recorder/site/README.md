# js-audio-plugin

纯 js 实现浏览器端录音。

## 支持功能

-   支持录音，暂停，恢复，和录音播放。
-   支持音频数据的压缩，支持单双通道录音。
-   支持录音时长、录音大小的显示。
-   支持边录边转（播放） 后续支持。
-   支持导出录音文件，格式为 pcm 或 wav。
-   支持录音波形显示，可自己定制。
